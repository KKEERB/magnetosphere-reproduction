# 车企配件采购计划半自动生成系统

## 项目简介

本系统是一个半自动化的配件采购计划生成工具，专为车企配件采购统计员设计。系统基于历史采购数据，通过数据驱动的预测模型自动计算采购量，并生成采购计划 Excel 和 Word 说明文档。

## 核心功能

- **多 Excel 文件合并**：自动读取和合并多个 Excel 数据文件
- **数据清洗和标准化**：处理缺失值、异常值、重复记录
- **ABC 优先级分类**：基于出库频次的智能分类（A/B/C/D/E 类）
- **需求预测模型**：自动选择最佳预测算法
  - 移动平均法 (SMA) - 稳定需求
  - 指数平滑法 - 有趋势需求
  - Croston 法 - 间歇性需求
- **采购量计算**：基于业务规则计算建议采购量
- **报告生成**：自动生成多 Sheet Excel 和 Word 说明文档

## 输入数据要求

### Excel 数据文件

放置在输入目录（默认：`~/.cc-connect/attachments/`），支持多个 Excel 文件自动合并。

**必需字段：**
- 配件号（或类似标识字段）
- 名称
- 单位
- 月度采购数据（格式：XX 年 X 月，如"23 年 1 月"）

### 业务规则文件（可选）

放置业务规则 Excel 文件（如"保内总库储备原则及标准.xlsx"），包含：
- 系数标准（ABC 分类规则、价格系数等）
- 数据逻辑（计算公式）
- 计算模型（参数配置）

### PMS 数据清洗规则（来自"当量制作.docx"）

系统自动应用以下清洗规则：

**1. 直供供应商标识过滤**
- 删除配件号后 5 位匹配直供供应商标识的记录
- 直供供应商标识：1218、7237、0155、4862、7479、2076、6393、F5693

**2. 配件类型过滤**
- 轮胎、桥总成、车架总成、柴油机总成、供气系统总成
- 滤芯、油品、磁铁、封胶、冷却液、发动机液、单瓶
- 变速箱、变速器等（变速箱单独储备）

**3. 关键词过滤（备注/订单类型）**
- KB 开头（如 KBFT0001786662025020365）
- 快报（质量快报、快报单号等）
- 整改（改制整改、整改报单等）
- 维修（维修报单、维修指导等，但维修厂不删）
- 专项订单、作业指导、保外转保内
- 招标出库、新产品方案、重点产品、星辉专用件等

**4. 订单状态过滤**
- 删除状态包含：撤单、新增、作废、终止 的记录

**5. 虚拟库过滤**
- 删除虚拟库记录：汇川、绿控、亿纬、空入空出

## 业务规则

### ABC 分类标准

| 分类 | 条件（12 个月出库频次 T） | 管理策略 |
|------|-------------------------|----------|
| A 类 | T ≥ 120 次 | 重点管理、精确预测 |
| B 类 | 36 ≤ T < 120 | 常规管理 |
| C 类 | 12 ≤ T < 36 | 简化管理 |
| D 类 | 1 ≤ T < 12 | 简化管理 |
| E 类 | T = 0 | 无需求、暂停储备 |

### 采购量计算公式

```
建议采购量 = MAX(0, 最大储备量 - 当前库存 - 在途库存)

最大储备量 = 日均需求 × 计划系数 × (备货周期 + 安全周期) / 30
```

### 预测模型选择

系统根据需求特征自动选择：
- **稳定需求** → 移动平均法
- **有趋势需求** → 指数平滑法
- **间歇性需求**（>50% 零需求）→ Croston 法

4. **文件路径**：避免路径中包含特殊字符，建议使用英文路径
